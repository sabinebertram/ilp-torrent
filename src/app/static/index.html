ILP Torrent

This page demonstrates the implementation of ILP torrent. ILP torrent is a tweaked version of WebTorrent, that uses a private tracker and the STREAM receipt verifier to check whether a license fee has been paid. The license is included in the torrent file on seeding. Hence, the payment pointer and the verifier endpoint are also part of the magnet URI.
